# ── Social preview crawlers ───────────────────────────────────────
# These bots fetch pages to build link-preview cards (OG / Twitter
# card / Slack unfurl / etc). When a social bot hits /film.html?id=X
# our Next.js middleware rewrites the request to /api/og/film-meta
# internally so the crawler gets dynamic og:image per film. Vercel
# surfaces the rewrite as an x-matched-path header — social bots
# then check robots.txt against that path, and the blanket
# Disallow: /api/ below used to trip the check (Facebook reported
